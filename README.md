# Currency Converter (Конвертер валют)

Простой GUI-проект на Python для конвертации валют с использованием внешнего API и сохранением истории в JSON.

## Описание

Приложение позволяет:
- выбрать валюту отправления и валюту получения;
- ввести сумму для конвертации;
- получить актуальный курс из внешнего API;
- сохранить историю конвертаций в файл `history.json`;
- загрузить историю из этого файла.

## Файлы проекта

- `converter.py` — основное приложение на Python с GUI на Tkinter.
- `requirements.txt` — зависимости проекта.
- `history.json` — файл истории конвертаций.
- `README.md` — это руководство.
- `.gitignore` — файлы и папки для игнорирования Git.

## Установка

1. Установите Python 3.10+.
2. Установите зависимости:

```bash
py -m pip install requests
```

## Как получить API-ключ

Приложение использует API `https://www.exchangerate-api.com/`.

1. Перейдите на сайт https://www.exchangerate-api.com/
2. Зарегистрируйтесь и получите бесплатный API-ключ.
3. Сохраните ключ в переменной окружения `EXCHANGE_API_KEY` или в файле `api_key.txt` в корне проекта.

Пример для Windows PowerShell:

```powershell
setx EXCHANGE_API_KEY "ВАШ_API_КЛЮЧ"
```

## Запуск

```bash
python converter.py
```

## Примеры использования

1. Выберите валюту `USD`, валюту `EUR`, введите сумму `100`.
2. Нажмите кнопку `Конвертировать`.
3. Внизу появится результат и запись в таблице истории.
4. Нажмите `Загрузить историю`, чтобы снова загрузить данные из `history.json`.
